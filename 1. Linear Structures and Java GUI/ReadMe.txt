File Descriptions:

1. LinkedList.java
	- Contains a generic linked list class

2. LinkedListNode.java
	- Contains a generic linked list node class

3. LinkedListTest.java
	- Contains JUnit tests for the LinkedList & LinkedListNode classes

4. DoublyLinkedList.java
	- Contains a generic doubly linked list class
	- Extends LinkedList

5. DoublyLinkedListNode.java
	- Contains a generic doubly linked list node class
	- Extends LinkedListNode

6. DoublyLinkedListTest.java
	- Contains JUnit tests for the DoublyLinkedList & DoublyLinkedListNode classes
	
7. RosterInfo.java
	- Contains the backend information for adding and deleting a student from both the waitlist and enrolled student lists.
	- Note: The current number of students allowed into the class is set to 5. This can change depending on class preference.
8. RosterDisplay.java
	- Contains the GUI elements stored in panels
9. RosterApplication.java
	- Contains the code to initialize the frame and create the GUI
